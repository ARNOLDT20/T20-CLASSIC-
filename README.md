## ⚡ Deployment Guide

### 1️⃣ Get Session ID
- Click the **Get Session ID** button above  
- Pair with your WhatsApp  
- Copy the generated Session ID  

### 2️⃣ Deploy to Heroku
- Click the **Deploy to Heroku** button above  
- Fill in:
  - `SESSION_ID`
  - `OWNER_NUMBER` (255xxxxxxxxxx)
  - `BOT_NAME` (Your choice)
- Deploy & Done ✅
